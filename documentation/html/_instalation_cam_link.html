Installation Instructions - CamLink - Multicam (Euresys)

NOTE: If you have a eBUS GEV interface go to >> Installation notes GEV - eBUS SDK (Pleora Technologies)

Installation notes for Euresys MultiCam SDK software The Euresys MultiCam SDK software provides a Camera Link compatible interface between the camera and a frame grabber card (Grablink Base, Grablink Full, etc), which may already be installed on the host PC with frame grabber card if PC setup was supplied by PSEL.

If PC / Grablink frame grabber card was not supplied and tested by PSEL please make sure to fit the frame grabber cards in their desired PCIe slots to ensure correct operation.
Recommended slot types for frame grabber cards:

  1. Grablink Base - PCIe x1 bus: 200 MB/s sustained delivery bandwidth
  2. Grablink Full - PCIe x4 bus: 850 MB/s sustained delivery bandwidth

Install the Euresys MultiCam SDK software by executing ‘multicam-6.9.8.2984.exe’. The MultiCam SDK install is first extracted and unzipped, then a window will be generated “Welcome to the MultiCam Setup Wizard”.

Click ‘NEXT’ and on the Setup Type screen please select ‘Full’ type and proceed on the next window via ‘NEXT’ button. In next window please leave the Destination Folder as default and continue to installation via ‘NEXT’ button, the installation which may take several minutes. After finish please restart your computer to load up the newly installed MultiCam driver.

NOTE: There is a known issue with the MultiCam driver which is reported by Euresys in their release notes (link) under Know Issues at page 11/15.

“Infrequent device start failure after cold boot. Affected boards: Grablink Base, Grablink DualBase, Grablink Full, Grablink Full XR Some cards may infrequently fail to start properly after power up (PC cold boot) and are not detected by the PC. The card operates properly again, after a power down / power up cycle of the PC.“

If your camera is not connecting / responding after a cold boot, please restart to allow the MultiCam driver to start up correctly.
